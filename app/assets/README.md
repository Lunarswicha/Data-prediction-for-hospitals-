# Assets — Logo et images

## Logo Pitié-Salpêtrière

Pour afficher le logo de l’hôpital dans le tableau de bord, placez ici un fichier nommé :

- **`logo_salpetriere.png`** (recommandé, fond transparent si possible)

ou

- **`logo_salpetriere.svg`**

Dimensions conseillées : largeur 200–300 px pour la sidebar, ou version plus grande pour l’en-tête.

Le logo AP-HP (dont dépend la Pitié-Salpêtrière) est disponible sur [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:AP-HP_Logo.svg). Vous pouvez également utiliser le visuel officiel du site [pitiesalpetriere.aphp.fr](https://pitiesalpetriere.aphp.fr) en respectant les droits d’usage.

Si aucun fichier n’est présent, le dashboard affiche un intitulé textuel à la place.
