# Application — Point d’entrée du prototype

Interface utilisateur du MVP (Streamlit, Dash ou autre) :

- Exploration des flux hospitaliers.
- Simulation de scénarios (épidémie, grève, afflux massif).
- Visualisation des prévisions (lits, personnel, matériel).
- Tableau de bord interactif + recommandations.

Consignes : « Interface permettant d’explorer les flux hospitaliers et de simuler différents scénarios » · « Modélisation des tendances d’admissions et prévision des besoins en lits, personnel et matériel médical ».
