# 03 — Modèles et résultats

Cette catégorie regroupe les documents relatifs aux **modèles de prédiction**, à leur **cohérence mathématique**, aux **métriques** et à l’**interprétation des résultats** (chiffres, pourcentages) affichés dans le dashboard.

| Document | Contenu |
|----------|---------|
| [COHERENCE-MODELES-PREVISION.md](COHERENCE-MODELES-PREVISION.md) | Inventaire des modèles (admissions, occupation, besoins), formules, cohérence, contradictions potentielles, complémentarités. |
| [EXPLICATION-POURCENTAGES.md](EXPLICATION-POURCENTAGES.md) | Signification des % affichés (taux d’occupation, % à ±10 % Bouteloup, seuils 85 %/95 %), pourquoi ils sont souvent élevés sur données synthétiques. |

**Justification méthodologique des modèles (niveau M2)** : voir [../01-rapport-conception/JUSTIFICATION-MODELES-PREDICTION.md](../01-rapport-conception/JUSTIFICATION-MODELES-PREDICTION.md).
